# cite.me.in Skills

AI agent skills for content teams. Install with [skillshare](https://skillshare.sh).

## Skills

| Skill | What it does |
|-------|-------------|
| [geo-blog-post](skills/geo-blog-post/SKILL.md) | Write blog posts optimized for AI citations (GEO) — structure, frontmatter, schema, and a pre-publish checklist based on Princeton research |

## Install

```bash
# Install all skills
skillshare install github.com/cite-me-in/skills --all

# Install a specific skill
skillshare install github.com/cite-me-in/skills -s geo-blog-post

# Track for future updates
skillshare install github.com/cite-me-in/skills --track --all
```

## What is GEO?

GEO (Generative Engine Optimization) means structuring content so AI engines — ChatGPT, Perplexity, Claude — cite it when answering user questions. These skills encode the patterns that consistently improve citation rates.
